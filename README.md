A premium, glassmorphism-styled Lunar Calendar web application featuring time, Gregorian and Lunar dates, zodiac info, and inspirational quotes.

## Features

- **Real-time Clock**: Displays current time with seconds in 24-hour format.
- **Gregorian & Lunar Calendar**: View the current month with integrated lunar dates.
- **Lunar Zodiac**: Automatically displays the current year's animal sign (e.g., Snake, Dragon).
- **Daily Quotes**: A curated collection of over 100 inspirational quotes that rotate daily.
- **Glassmorphism Design**: Modern, premium aesthetic with smooth gradients and blurred backgrounds.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd lunar-calendar
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Start the development server:
```bash
npm run dev
```

Open your browser to the URL shown in the terminal (usually `http://localhost:5173`).
